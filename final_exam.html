Learning goals

  • Use all topics from the 7-day plan.
  • Work independently under exam conditions.
  • Check answers using inverse operations and estimation.

Mini lesson

Read each question carefully, choose the correct method, calculate, and then check whether the answer is sensible. For mixed exams, identifying the topic is often the most important first step.

Worked examples

  1. If a question asks for change, subtract the cost from the amount paid.
  2. If a question asks for area of a rectangle, multiply length by width.
  3. If a division question has no remainder, use multiplication to check.
