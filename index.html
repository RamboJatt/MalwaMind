How to use

  1. Sign in with a learner name.
  2. Open a day below.
  3. Type answers in the boxes and use Check all or check one question separately.
  4. Use the admin area to view each learner’s entered answers next to the correct answers.

This local HTML pack stores results in the same browser on the same computer using local storage.
