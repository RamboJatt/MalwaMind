Year 5 Maths Interactive Daily Pack

Split into daily HTML pages. Learners can type answers directly on the computer, check their score, reveal answers, and save progress in the browser.

How to use

  1. Open a day below.
  2. Type answers in the boxes.
  3. Use Check all or check each question separately.
  4. On Day 7, complete the mixed revision page and then the final exam.

Progress is saved only in the same browser on the same computer. Each page also has a print button.
