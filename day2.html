Study material

Learning goals

  • Use formal written column methods for addition and subtraction.
  • Estimate answers and check using inverse operations.
  • Solve multi-step word problems.

Mini lesson

Line up digits by place value before adding or subtracting. Use estimation to check whether your answer is sensible. Addition and subtraction are inverse operations, so you can check 54,203 − 18,944 = 35,259 by doing 35,259 + 18,944.

Worked examples

  1. 23,481 + 8,736 = 32,217.
  2. 70,000 − 28,456 = 41,544.
  3. If a shop has 4,500 items, sells 1,275 and receives 850 more, it has 4,075 items.
